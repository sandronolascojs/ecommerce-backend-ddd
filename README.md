
## eCommerce backend 

# Tecnologies

- NodeJS
- TypeScrypt
- Prisma ORM
- PostgreSQL
- JsonWebToken
- Jest

# Functions

- Customer CRUD
- Refresh Token 
- orders CRUD
- order Items CRUD
- Products CRUD
- Categories CRUD
- Sub categories CRUD

# Routes

- /api/v1/customers
- /api/v1/addresses
- /api/v1/administrators
- /api/v1/orders
- /api/v1/orderItems
- /api/v1/categories
- /api/v1/subcategories
- /api/v1/authentication
  
# How to run

- Clone the repository
- Add environments Variables
- Run a PostgresSQL database

# Environments Variables

- PORT
- DATABASE_URL
- CORS_ORIGIN
- JWT_SECRET

# Commands
- npm/pnpm/yarn install
- npm/pnpm/yarn run dev
- npm/pnpm/yarn build
- npm/pnpm/yarn start
